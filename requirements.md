## 1. Транспорт (Bus Route)

### Описание задачи
Необходимо реализовать систему маршрута автобуса, включающую список остановок.
Каждая остановка содержит:
- название
- координаты
- время до следующей остановки

### Обязательная часть
Реализовать следующие функции:
- Добавление остановки
- Расчет общего времени маршрута
- Определение, где будет автобус через N остановок
- Построение обратного маршрута

### Вариативная часть (на выбор)
Реализовать **одну или несколько** функций:

1. Реализовать кольцевой маршрут

*2. Найти остановки, до которых автобус доедет за заданное время*

3. Добавить расчет времени прибытия (расширение функции "где будет автобус через N остановок")

*4. Создать функцию формирования подробного отчета маршрута с сохранением в текстовый файл*

5. Вывести маршрут в виде таблицы с выравниванием колонок
6. Сохранить маршрут в текстовый файл, где каждая строка — Base64-кодированная информация об остановке
